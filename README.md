# MDP.AspNetCore.AdminHub

MDP.AspNetCore.AdminHub是開源的.NET開發套件，協助開發人員快速建立具有後台管理的應用系統。提供身分驗證、角色授權、選單管理等多種功能模組，用以簡化開發流程並滿足多變的商業需求。
